# Ralph Progress Log - Derby Digital Onboarding Portal

## Codebase Patterns
- Brand colors: derby-dark (#0A0A0F), derby-card (#1A1A2E), derby-blue (#2093FF), derby-blue-deep (#0026FF)
- Use `@/components/wizard/WizardContext` for step state — useWizard() hook provides currentStep, formData, navigation
- STEPS array and TOTAL_STEPS constant exported from WizardContext for step metadata
- Wizard animations use CSS keyframes in globals.css (animate-step-in, animate-step-in-reverse)
- Page is "use client" — WizardProvider wraps WizardLayout which wraps StepRenderer
- Mobile responsive: sm: breakpoint for desktop step indicators, mobile shows simplified "Step X of Y"
